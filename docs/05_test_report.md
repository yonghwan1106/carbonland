## 2. 상세 테스트 결과

### 2.1 메인 페이지 (/)

**정상 동작:**
- 헤더: CarbonLand 로고, 해커톤 정보, About/GitHub 링크
- 지도: 경기도 중심으로 OpenLayers 지도 렌더링
- 컨트롤 패널: 데모 영역 5개, 영역 그리기 버튼, 토지이용 선택
- 결과 패널: 시나리오 비교 차트, 시뮬레이션 결과
- 푸터: 데이터 출처 표시

### 2.2 지도 기능

**WMS 레이어 (경기기후플랫폼 API):**
| 레이어 | WMS명 | 상태 |
|--------|-------|------|
| 탄소흡수지도 | spggcee:rst_npp | ✅ |
| 수목탄소저장 | spggcee:plnt_cbn_strgat_biotop | ✅ |
| 토양탄소저장 | spggcee:rst_soil_cbn_strgat_32652 | ✅ |
| 비오톱유형도 | spggcee:biotop_lclsf | ✅ |

**영역 선택:**
- 프리셋 5개 지역 모두 정상 동작
- 다각형 그리기: 클릭으로 꼭지점, 더블클릭으로 완료
- 면적 계산: Turf.js 사용, ha/m² 단위 표시

### 2.3 시뮬레이션

**토지이용 유형 (7종):**
- 산림지, 초지/공원녹지, 농경지, 습지
- 주거지, 상업지, 공업지

**시나리오 (5종):**
- 주거지 개발, 상업지 개발, 공업지 개발
- 녹지 보전, 산림 복원

**계산 항목:**
- 즉시 배출량 (토지 전환 시)
- 연간 흡수량/배출량 변화
- 누적 탄소 변화 (N년)
- CO2 환산값

### 2.4 결과 시각화

**차트:**
- 시나리오 비교 막대 차트 (Recharts)
- 연도별 탄소 변화 선 차트
- Before/After 비교 차트

**환산 지표:**
- 30년생 소나무 환산 (그루)
- 승용차 연간 배출량 환산 (대)
- 가구 연간 배출량 환산 (가구)

### 2.5 모바일 반응형

- 하단 탭 네비게이션 (지도/설정/결과)
- 슬라이드업 패널 UI
- 터치 친화적 버튼 크기

## 5. 결론

**전체 평가: 정상 동작**

MVP 기능이 모두 구현되어 있으며, 경기기후플랫폼 API 연동이 완료되었습니다.
