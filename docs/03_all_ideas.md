## 아이디어 1: 기후위기 복합재해 조기경보 시스템

### 개요
폭염 + 산사태 + 홍수 데이터를 통합하여 복합재해 위험도를 실시간 분석하고, 취약계층(노인, 어린이)에게 맞춤형 대피 경로를 제공하는 웹앱

### 사용 데이터
- 폭염 등급 평가, 체감온도
- 산사태 위험등급
- 극한호우 위험도 지수
- 무더위쉼터, 이재민 임시거주시설 위치
- 폭염취약지역 생활인구

### 차별점
단일 재해가 아닌 **복합재해 시나리오** (예: 폭염 + 집중호우 동시 발생) 대응

### 정책 임팩트
경기도 31개 시군 재난안전대책본부에서 바로 활용 가능

### 평가
- 기술 완성도: ★★★★☆
- 정책 임팩트: ★★★★★
- 혁신성: ★★★★☆

## 아이디어 2: 탄소중립 토지이용 시뮬레이터 ⭐ 선정

### 개요
개발/보전 계획에 따른 탄소 흡수/저장/배출 변화를 시뮬레이션하는 의사결정 지원 도구

### 사용 데이터
- 탄소흡수지도 (10m, 비오톱)
- 수목 탄소저장지도
- 토양 탄소저장지도
- 건축물 탄소배출량
- 비오톱 유형도

### 기능
- 특정 구역 선택 → 현재 탄소 수지 계산
- "이 녹지를 개발하면?" 시나리오 시뮬레이션
- 탄소중립 달성을 위한 대안 토지이용 제안

### 정책 임팩트
도시계획, 환경영향평가 시 정량적 근거 제공

### 평가
- 기술 완성도: ★★★★☆
- 정책 임팩트: ★★★★★
- 혁신성: ★★★★★

## 아이디어 3: 태양광 최적 입지 AI 매칭 플랫폼

### 개요
건물 옥상/유휴 부지의 태양광 설치 가능성을 자동 분석하고, 규제 요인 검토 및 예상 수익을 계산해주는 서비스

### 사용 데이터
- 태양광 잠재량 (이론적/기술적/시장)
- 19종 규제요인 데이터 (국립공원, 문화재보호구역, 상수원보호구역 등)
- 건물정보
- 태양광 발전 현황/설치 예정지

### 기능
- 주소 입력 → 자동 규제 체크 + 잠재량 분석
- ROI(투자수익률) 계산기
- 유사 설치 사례 비교

### 정책 임팩트
경기 RE100 달성 가속화, 주민 참여형 에너지 전환

### 평가
- 기술 완성도: ★★★★★
- 정책 임팩트: ★★★★☆
- 혁신성: ★★★☆☆

## 아이디어 4: 공원 형평성 분석 대시보드

### 개요
경기도 4,120개 도시공원의 접근성 격차를 시각화하고, 녹지 소외지역을 발굴하는 데이터 대시보드

### 사용 데이터
- 공원 서비스 종합평가 (시군/읍면동)
- 생태계서비스 스코어
- 폭염취약지역 생활인구
- 녹지 현황도, 공원 현황도

### 기능
- 시군별 공원 접근성 랭킹
- "녹지 사각지대" 히트맵
- 신규 공원 조성 시 기대효과 시뮬레이션

### 정책 임팩트
환경정의(Environmental Justice) 관점의 도시계획 지원

### 평가
- 기술 완성도: ★★★★☆
- 정책 임팩트: ★★★★☆
- 혁신성: ★★★★☆

## 아이디어 5: 그린리모델링 우선순위 추천 시스템

### 개요
건축물별 에너지 소비/탄소배출 데이터를 분석하여 그린리모델링 우선 대상 건물을 추천하고 예상 절감량을 제시

### 사용 데이터
- 건물정보 (에너지 소비, 탄소배출량)
- 폭염 등급 평가 (열환경)
- 비오톱 토지피복

### 기능
- 건물 유형별 에너지 효율 벤치마크
- 리모델링 우선순위 점수 산정
- 예상 탄소 감축량 및 비용절감 계산

### 정책 임팩트
공공건축물 에너지 효율화 정책 수립 지원

### 평가
- 기술 완성도: ★★★★☆
- 정책 임팩트: ★★★★☆
- 혁신성: ★★★☆☆
